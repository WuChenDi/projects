## Description

Add lightweight tag-based organization to links (today only a free-text
`comment` exists).

- `database/schema.ts` — add a `tags` JSON column (`string[]`, default `[]`) to
  `links`; emit an **incremental** migration via `pnpm --filter @cdlab/sink
  db:gen` (tool-generated, never hand-edited).
- `schemas/link.ts` — add a bounded `tags` array to create/edit/import; thread
  through `lib/links.ts`.
- Editor — tag chip input (add/remove).
- List (`links-view.tsx`) — render tag chips + a tag filter (default to simpler
  client-side filtering on the loaded page; server-side `LIKE` if needed).

## Acceptance Criteria

Tags persist on create/edit, survive export/import, render as chips in the list,
and are filterable; one additive migration generated by drizzle-kit; existing
rows default to `[]`; build + biome clean; en/zh i18n.
