# FEAT-023 flnk — launchpads: data model + CRUD API

- **status**: completed

## Description

Introduce **launchpads** — hosted marketing / link-in-bio pages managed in the
dashboard and published at `/m/<slug>`. This task lands the data layer and the
authenticated CRUD API only; public render (FEAT-024) and editor (FEAT-025)
build on it.

The launchpad is the management-side entity; `/m/<slug>` is its public published
view. A launchpad owns an ordered list of content **blocks**.

## Schema (new table)

`launchpads`:

- `id` (pk, genid), `slug` (global unique — `/m/<slug>` resolves by URL with no
  auth context), `ownerId` (= `user.id`, **stamped on write, not yet filtered**
  per the baseline discipline), `title` (display name).
- `status`: `'draft' | 'published'` (default `'draft'`). `/m/<slug>` serves
  not-found unless `published`.
- `config` (JSON): `{ profile: { avatar?, name?, bio? }, theme: { preset,
  primaryColor, buttonShape }, blocks: Block[] }`.
- `og`: page-level `{ title?, description?, image? }` for sharing `/m/<slug>`.
- `expiresAt` (optional), + shared `trackingFields`.

`Block` union (config.blocks, order = render order, each `{ id, type, enabled }`):
`header` (profile) · `socials` (platform+url list) · `button` (label + target:
short-link id or url) · `shortlink` (referenced link id list) · `image`
(src + optional link) · `text` · `divider`.

`shortlink` / `button`-to-short-link targets store the **link id (reference)**;
clicks go through `/<slug>` so the short link's own stats and editable
destination are reused.

## API (`/api/launchpad/*`, all `requireSession` + ownerId stamp)

`create` · `edit` · `upsert` · `delete` (soft) · `list` · `query` · `publish`
(flip status). Owner filtering is routed through a single `scopeToOwner(query,
session)` helper that is a **no-op now** (shared workspace) and becomes the
isolation switch later.

## Acceptance Criteria

- Migration generated by drizzle-kit and applied locally.
- CRUD round-trips a launchpad incl. blocks; slug global-unique enforced; soft
  delete respected.
- Every row carries `ownerId` from the session; `scopeToOwner` present but no-op.
- `m` reserved (cannot be created as a short-link slug).
- `tsc` + biome clean.
